# ONION — офлайн заметки

Ваш проект `onion_notes` (тот, что вы создали через `flutter create`)
теперь заполнен рабочим приложением: карточки на главном экране, поиск по
названию/дате, редактор с чекбоксами (анимированная галочка + зачёркивание),
жирным/курсивом/маркером/заливкой по выделению, вставкой фото со
скруглёнными углами, анимированными фонами, историей назад/вперёд,
переносимым текстовым "кодом" заметки и экраном «Дороги» — холст, где
заметки соединяются нитями в квест-цепочки.

## Иконка приложения

Загруженная вами иконка (фиолетовый блокнот с луковицей) уже установлена:
- как иконка приложения на главном экране телефона — файлы разложены по
  `android/app/src/main/res/mipmap-*/ic_launcher.png` (mdpi/hdpi/xhdpi/
  xxhdpi/xxxhdpi, каждый в своём размере);
- как иконка веб-версии (`web/icons`, `web/favicon.png`), на случай если
  когда-нибудь соберёте `flutter build web`;
- внутри самого приложения — она отображается в шапке главного экрана
  рядом с надписью «ONION» (файл `assets/icon/app_icon.png`).

Менять ничего не нужно — просто соберите APK, иконка будет уже там.

## Шрифты — важный нюанс

В моей песочнице нет интернета, поэтому реальные файлы шрифтов (Poppins,
Inter и т.д.) физически невозможно было скачать и положить в проект.
Чтобы не сломать вам сборку ссылкой на несуществующие файлы, сейчас
приложение использует системные шрифты Android (обычный / с засечками /
моно / рукописный) — они работают из коробки, без единого лишнего файла.

Если хотите настоящие Poppins/Inter/Caveat и т.д.:

```bash
cd ~/onion_notes   # или как называется ваша папка с проектом
chmod +x scripts/download_fonts.sh
./scripts/download_fonts.sh
```

Затем раскомментируйте блок `fonts:` в `pubspec.yaml` (там есть пометка
где именно) и в `lib/screens/editor_screen.dart` замените строку
`const _fontChoices = ['', 'serif', 'monospace', 'cursive'];` на
`const _fontChoices = ['Poppins', 'Inter', 'PlayfairDisplay', 'Caveat', 'JetBrainsMono'];`
(комментарий прямо над этой строкой подсказывает то же самое).

## Сборка через GitHub Actions

```bash
flutter pub get      # проверить, что всё подтянулось
git init
git add .
git commit -m "ONION notes app"
git branch -M main
git remote add origin https://github.com/<логин>/onion_notes.git
git push -u origin main
```

`.github/workflows/build_apk.yml` соберёт APK автоматически при пуше
(и по команде `gh workflow run build_apk.yml`), с иконкой уже внутри.
Готовые файлы — во вкладке Actions → артефакт `onion-notes-apk`, либо:

```bash
pkg install gh   # в самом Termux, не в proot
gh auth login
gh run list --workflow=build_apk.yml
gh run download <RUN_ID> -n onion-notes-apk -D ~/storage/downloads/
```

## Локальная сборка (по желанию)

```bash
flutter pub get
flutter build apk --release
```

## Как устроен "код заметки"

Кнопка ✨ в редакторе → «Скопировать код заметки» — получаете обычный
текст вида `#ONION-NOTE-V1 ...`, который можно отправить куда угодно.
На другом телефоне: Настройки → «Вставить заметку из кода» — всё
восстановится, включая чекбоксы, стили и фото (base64 внутри кода).
